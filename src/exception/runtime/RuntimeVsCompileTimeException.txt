What is Compile Time Excepiton ?
Compile Time Exception are those which compiler forces us to handle it at compile time.
Example of Compile Time exception are -
* IOException
* SQLException
* FileNotFoundException
* ClassNotFoundException

What is Runtime Exception ?
A RuntimeException is an unchecked exception in Java â€” it occurs at runtime and is not checked by the compiler.
Example of Runtime exception are -
* NullPointerException
* ArrayIndexOutOfBoundsException
* ArithmeticException
* ClassCastException
